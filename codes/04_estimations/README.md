# 04_estimations

This folder contains all the codes used for the estimations of the project. The files should be executed in the order listed below so that each script uses the output generated by the previous step.

- **010_Data_prep.R:** prepares the combined baseline and endline datasets. It fixes the format of some variables, adds categories for mental and physical health indicators and merges everything into a single dataset for the next step.
- **011_Propensity_Score_Matching.R:** uses the dataset created in `010_Data_prep.R` and performs the propensity score matching estimation. The script relies on the nearest method with a logit link, a glm distance, replacement and a ratio of 1:5 for treated and control households. It also produces balance plots and tables and saves both cross-sectional and panel matched datasets.
- **012_Differences_in_differences.R:** takes the matched datasets and runs all the differences-in-differences estimations. Results are produced by variable group and exported to LaTeX tables. Most groups are estimated with the panel data (mental health, physical health, labor outcomes, family environment, resignification of space, community environment and income outcomes). Some relocation variables use the cross-sectional data from the endline only. Additional estimations explore perceptions about private firms using city subsamples.

There are three additional folders that contain the following codes:
- **old:** has additional codes that were used at some point but are no longer relevant to the current estimations. This includes different specifications for the regressions and lasso reduction.
- **src-aux:** contains one additional code that creates a general balance table and is used as a table template for the estimation codes.
- **women_households:** contains the same data preparation, propensity score matching and differences in differences estimations but only using the households that are lead by women.
